# Git Commits (How to)

1. Stage changes.

   ```sh
   git add .
   ```

2. Commit changes with a description.

   ```sh
   git commit -m"<related_message_about_staged_files_changes>"
   ```

3. Check Git status

   ```sh
   git status
   ```

4. Push to origin/master

   ```sh
   git push origin master
   ```

5. Pull code from master
   ```sh
   git pull
   ```
